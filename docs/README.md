# Dokumentacija

Ta mapa vsebuje dokumentacijo predavanj, vaj in projektnih nalog.

## Struktura

```
docs/
├── predavanja/     # Zapiski s predavanj
├── vaje/           # Gradiva in rešitve vaj
└── projekti/       # Projektna dokumentacija
```

## Navodila

Vsak dokument naj vsebuje:
- Datum nastanka
- Temo / naslov
- Ključne ugotovitve
- Vire
