# Domače naloge in projekti

Ta mapa vsebuje domače naloge, rešitve in projektne oddaje.

## Struktura

```
assignments/
├── dn01/     # 1. domača naloga
├── dn02/     # 2. domača naloga
└── projekt/  # Projektna naloga
```

## Navodila

Za vsako nalogo ustvari svojo podmapo in vanjo dodaj:
- Navodila naloge (če so dostopna)
- Rešitev
- Kratko poročilo ali razlago pristopa
