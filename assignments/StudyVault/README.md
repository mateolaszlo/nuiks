# StudyVault

StudyVault is a microservice-based MVP for course work at NUIKS. The repository is organized as a monorepo with a React frontend, FastAPI backend services, shared packages, infrastructure config, and automated tests.

## Structure

- `apps/` runnable applications
- `packages/` shared code
- `infra/` Docker, gateway, identity, and observability config
- `tests/` automated tests across unit, service, integration, and smoke layers
- `docs/` supporting project documentation
- `studyvault_mvp_requirements.md` MVP scope and feature requirements

## Planned Services

- `frontend`
- `gateway`
- `file-service`
- `catalog-service`
- `search-service`
- `activity-service`
